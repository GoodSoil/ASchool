Aschool

ASchool is a system for managing courses and planning course selections for a self-registration system. It's also an exploration in handling (somewhat) complex course dependency and registration requirements.

Download this project as a .zip file Download this project as a tar.gz file

Housed at GitHub.

This project is housed at GitHub and owned/managed by @dagilleland. It has been birthed out of a perceived need for something like this for students and out of a desire to just improve my own programming skills and abilities.

Authors and Contributors.

At present, this project is open to contributors among people I know and/or work with. Right now, I've got about four people interested in helping out, but I might have room for about two or three more. As time goes on and requirements grow/change, I expect the list of contributors to change.

Expectations of Contributors.

Early in the life of this project, I ran across an online book titled "Designing Evolvable Web APIs with ASP.NET", and my hopes are this book will be a great resource for building the ASchool project. In fact, it is required reading for anyone planning on contributing to this project. :)
