Aschool

ASchool is a system for managing courses and planning course selections for a self-registration system. It's also an exploration in handling (somewhat) complex course dependency and registration requirements.

Download this project as a .zip file Download this project as a tar.gz file

Housed at GitHub.

This project is housed at GitHub and owned/managed by @dagilleland. It has been birthed out of a perceived need for something like this for students and out of a desire to just improve my own programming skills and abilities.
